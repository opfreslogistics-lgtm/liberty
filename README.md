# Liberty National Bank - Online Banking Platform

A modern, minimalist, and highly responsive online banking platform built with Next.js.

## Features

- **Mobile-First Design**: Responsive layout with persistent bottom navigation on mobile and sidebar on desktop
- **Account Management**: View balances, account details, and spending summaries
- **Transfers & Payments**: Internal transfers, external ACH, and peer-to-peer payments
- **Card Management**: Virtual card display with instant controls
- **Transaction History**: Advanced search and filtering
- **Crypto Trading**: BTC integration with real-time rates
- **Loans & Credit**: Loan status and application forms
- **Budgeting & Planning**: Category-based budgeting and savings goals
- **Multi-Currency**: Support for multiple currencies with FX conversion
- **Security**: Two-factor authentication, biometric login support
- **Dark Mode**: Elegant theme toggle

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Zustand (State Management)
- Recharts (Data Visualization)
- Lucide React (Icons)
